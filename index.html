母音 (모음)

ㅏ ㅑ ㅓ ㅕ ㅗ ㅛ ㅜ ㅠ ㅡ ㅣ

韓国語には10個の基本母音があります。

ㅏ = a（아）、ㅓ = eo（어）、ㅗ = o（오）、ㅜ = u（우）、ㅡ = eu（으）、ㅣ = i（이）

それぞれの母音は子音と組み合わせて音節を作ります。

子音 (자음)

ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅅ ㅇ ㅈ ㅎ

韓国語には14個の基本子音があります。

ㄱ = k/g、ㄴ = n、ㄷ = t/d、ㄹ = r/l、ㅁ = m、ㅂ = p/b、ㅅ = s、ㅇ = ng/silent、ㅈ = j/ch、ㅎ = h

子音と母音を組み合わせて文字を作ります。

音節の組み合わせ

가 나 다 라 마 바 사

ハングルの音節は「子音 + 母音」または「子音 + 母音 + 子音」の形で作られます。

例：ㄱ + ㅏ = 가（ga）、ㄴ + ㅏ = 나（na）、ㄷ + ㅏ = 다（da）

練習：한국어（ハングゴ）= 韓国語

📝 文法

1 基本文型
2 助詞

基本文型

韓国語の語順：主語 + 目的語 + 動詞

例：나는 한국어를 배워요。

naneun hangugeoreul baeweoyo
私は韓国語を学んでいます。

です・ます体：입니다 / 합니다

丁寧な表現に使います。

例：학생입니다（학생이다）= 学生です

助詞

は：은 / 는

子音終わり → 은、母音終わり → 는

例：나는（私は）、학교는（学校は）

が：이 / 가

子音終わり → 이、母音終わり → 가

例：사람이（人が）、나가（私が）

を：을 / 를

子音終わり → 을、母音終わり → 를

例：책을（本を）、음악를（音楽を）
